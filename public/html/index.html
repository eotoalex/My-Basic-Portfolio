In the GitHub, I usually state my process, where I see the app going and/or what is giving me issues while building.

Recently used the MERN stack, coupled with a City Crime API and the Google Maps API to create an interesting app. The app we are currently working on ,Trekky, tracks crime data and user locations on a map. A lot of my colleagues tell me it is just like the Citizen app, but my data is historical data not real-time data. We are working on creating a tracking/cataloging system for crime around the city as this application grows, so it wont only be used by citizens but by law enforcement as well.
